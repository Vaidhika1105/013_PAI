CASE STUDY CSP: Online Exam Proctor Assignment using CSP (Backtracking)

CODE:

# Step 1: Define proctors and exams
proctors = ["Alice", "Bob", "Charlie"]
exams = ["Math", "Physics", "Chemistry"]

# Step 2: Store assignments
assignment = {}

# Step 3: Constraint check
def is_valid(proctor):
    # Ensure proctor is not already assigned
    return proctor not in assignment.values()

# Step 4: Backtracking function
def assign_proctors(index=0):
    
    # If all exams are assigned
    if index == len(exams):
        return True
    
    # Try each proctor
    for p in proctors:
        if is_valid(p):
            # Assign proctor
            assignment[exams[index]] = p
            
            # Recur for next exam
            if assign_proctors(index + 1):
                return True
            
            # Backtrack
            del assignment[exams[index]]
    
    return False

# Step 5: Run program
assign_proctors()

# Step 6: Output result
print("Exam Proctor Assignment:", assignment)


OUTPUT:

Exam Proctor Assignment: {'Math': 'Alice', 'Physics': 'Bob', 'Chemistry': 'Charlie'}
